Commands

Moderation:

r!purge [1-100] Delete amount of messages
r!ban [@user] [reason] ban user
r!banid [UserID] [reason] ban user using ID
r!kick [@user] kick user
r!warn [@user] [reason] warn user
r!warnings [@user] Display warnings of specific user
r!clearwarns [@user] [reason] Clear all warnings of specific user
r!mute [@user] Mute user
r!unmute [@user] Unmute user
r!voice-mute [@user] Voice mute user
r!voice-unmute [@user] Voice unmute user
r!deafen [@user] Deafen user
r!undeafen [@user] Undeafen user
r!votekick [@user] [reason] Vote to kick a user (Won't make the kick)

Setup:

r!mod-log [#channel / off] set mod-log or turn mod-log off
r!server-log [#channel / off] set server-log or turn server-log off
r!invite-filter [on/off] Turn on or off the invite links filter
r!links-filter [on/off] Turn on or off the links filter
r!mutedrole Create the 'Muted' role

Other:

r!invite Invite rick to your server
r!about Info about Rick
r!stats Rick's Stats
r!uptime Rick's Uptime
